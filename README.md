# My-Ecommerce-Website
A full-stack eCommerce website built with Python, featuring product listings, user authentication, a shopping cart, secure checkout, and order management. The project includes a responsive user interface and structured backend, demonstrating clean code practices, database integration, and core features required for a modern online store.

## Features
- User authentication (registration, login, logout)
- Product listings with detailed descriptions
- Search and filter functionality for products
- Shopping cart with quantity management
- Secure checkout process with payment integration
- Order history and management for users
- Admin panel for managing products and orders
- Responsive design for mobile and desktop devices
